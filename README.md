# Overview

Project Lumina aims to reverse engineer the factory illuminated Volkswagen front emblem found on the MEB platform.

The objective is **not simply to install an aftermarket logo**, but to understand and document the original Volkswagen implementation—including wiring, control modules, electrical behaviour, OEM components, and installation methods.

Every discovery in this repository is based on one of three categories:

- ✅ Confirmed facts
- 🟡 Working theories
- 🔬 Measured data

The goal is to separate assumptions from verified engineering information.

# Project Goals

- Document the OEM illuminated emblem system
- Identify all required OEM components
- Locate the correct J519 output
- Build an OEM-compatible wiring solution
- Create a complete installation guide
- Publish verified measurements
- Document every engineering discovery
